# 행동 강령

모든 참여자는 경험, 성별, 장애, 국적, 인종, 종교, 정체성과 관계없이 존중받아야
합니다. 기술적 의견 차이는 근거와 재현 가능한 결과를 중심으로 논의합니다.

다음 행동은 허용하지 않습니다.

- 괴롭힘, 차별, 모욕, 위협과 지속적인 비생산적 방해
- 동의 없는 개인정보 공개와 사적 연락의 공개 전재
- 고의적인 비밀정보 노출, 데이터 권리 침해 또는 서비스 악용
- 운영자가 중단을 요청한 뒤에도 계속되는 부적절한 행동

운영자는 상황에 따라 콘텐츠 수정 요청, 경고, 일시적 참여 제한 또는 영구 차단을
적용할 수 있습니다. 민감한 신고는 공개 Issue 대신 [SECURITY.md](SECURITY.md)의
비공개 경로를 사용합니다. 운영자는 필요한 범위에서만 정보를 공유합니다.

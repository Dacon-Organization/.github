# Dacon Organization

DACON 대회 참가와 데이터·AI 프로젝트를 독립 저장소로 관리하는 개인 학습
Organization입니다. **DACON 플랫폼의 공식 운영 조직이 아닙니다.**

## 프로젝트

| 저장소 | 설명 | 상태 |
| --- | --- | --- |
| [Dacon](https://github.com/Dacon-Organization/Dacon) | 기존 통합 저장소와 이전 기록 | 이관 완료 |
| AlphaFolio | 투자 대시보드 해커톤 | 분리 준비 중 |
| QualityLens | 스마트 팩토리 품질 분석 | 분리 준비 중 |
| JB RegTwin Local | 금융 AI 규제 분석 | 분리 준비 중 |
| News Health Check | 뉴스 데이터 분석 | 분리 준비 중 |
| BARAM 2026 | 풍력 발전량 예측 | 분리 준비 중 |

모든 변경은 개인 브랜치, PR, 공통 CI와 squash merge 흐름을 따릅니다. 현재는 solo
모드이며 팀 프로젝트 전환 시 저장소별 승인자와 CODEOWNERS를 별도로 설정합니다.

직접 작성한 코드와 문서는 MIT License를 사용합니다. 데이터와 기타 제3자 자료에는
원 출처의 이용 조건이 적용됩니다. 기여 방법은
[공통 가이드](https://github.com/Dacon-Organization/.github/blob/main/CONTRIBUTING.md),
보안 신고는 [보안 정책](https://github.com/Dacon-Organization/.github/security/policy)을
참고하세요.

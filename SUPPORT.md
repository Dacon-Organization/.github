# 지원 안내

- 재현 가능한 오류는 **Bug report** Issue Form을 사용합니다.
- 개선 제안은 **Feature request** Issue Form을 사용합니다.
- 데이터 출처, 권리 또는 무결성 문제는 **Data report** Issue Form을 사용합니다.
- 보안 취약점과 비밀정보는 공개 Issue 대신 [SECURITY.md](SECURITY.md)를 따릅니다.

프로젝트는 개인 대회 참가와 학습 목적으로 유지되며 지원 시간이나 응답 기한을
보장하지 않습니다. 문제를 빠르게 파악할 수 있도록 대상 commit, 환경과 최소 재현을
포함해 주세요.
